CSS Text Properties

i)color                                                                                           :red,green,blue..etc
 Sets the color of text

ii)text-align                                                                                     :left,right,center,justify                                             
Specifies the horizontal alignment of text

iii)text-decoration                                                                               :overline,underline,line-through,none 
The text-decoration property is used to set or remove decorations from text.
 
 We can use below one's
    text-decoration-line (required)                                                               : overline,underline,line-through,none 
    text-decoration-color                                                                         :red,green,blue..etc
    text-decoration-style                                                                         :solid,double,dotted,dashed,wavy

 text-decoration: text-decoration-line text-decoration-color text-decoration-style|initial|inherit;

 eg :  text-decoration: overline  red dashed;

 iii)text-transform                                                                               :lowercase,uppercase,capitalize,none
The text-transform property is used to set the cases for a text.

iv) text-indent                                                                                   :length
The text-indent property is used to set the indentation of the first line of text within a block of text. It is typically done by inserting the empty space before the first line of text.
 
v)text-overflow                                                                                   :clip,ellipsis
Specifies how overflowed content that is not displayed should be signaled to the user

vi)text-shadow
Specifies the shadow effect added to text 
text-shadow: *left-offset* *top-offset* *blur-distance* *color*;
eg :text-shadow: 2px 2px 1px #000000;

vii)direction                                                                                      :ltr,rtl            
The direction property specifies the text direction/writing direction within a block-level element.

viii)line-height                                                                                   :length,normal,number
Sets the line height

ix) word-spacing                                                                                   :number                                                         
The word-spacing property increases or decreases the white space between words.

x)letter-spacing                                                                                   :number    
Increases or decreases the space between characters in a text

xi)white-space                                                                         :normal,nowrap,pre,pre-line,pre-wrap
 Specifies how white-space inside an element is handled

xii)vertical-align                                                                    : baseline,length,sub,super,top,text-top,middle,bottom,text-bottom
Sets the vertical alignment of an element
